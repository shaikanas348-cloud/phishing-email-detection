# Phishing Email Detection System

A Machine Learning based Phishing Email Detection System built using Python, FastAPI, and Streamlit.

## Features
- Detects phishing emails
- Machine Learning based classification
- FastAPI backend
- Streamlit web interface

## Tech Stack
- Python
- Scikit-learn
- FastAPI
- Streamlit
- Pandas
- NumPy
